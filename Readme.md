# Convert images to PDF

## How to use

```commandline
Script to convert images to pdf

HOW TO USE:

    convert_img_to_pdf image.png
    convert_img_to_pdf .
    convert_img_to_pdf /home/user/Pictures 
```

## Dependence

- Pillow

### Install dependence

First step create a venv and then install dependence

```commandline
pip install Pillow
```

Or

```commandline
pip install -r requirements.txt
```
